Look up an address

Pattern principals

  • search for addresses from a data register that is maintained by a third party
  • collect addresses in approved formats that can be used for straight-through backend processing
  • allow address to be search by postcode or the first line
  • allow manual edit/entry if the address is not held in the way the user expects
  • manually edited/entered addresses cannot be processed automatically
  • manually edited/entered address should trigger a request a verify and update data register if required.
